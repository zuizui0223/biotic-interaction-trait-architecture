# Raw host-evidence intake

Do not place downloaded PDFs, screenshots, or unverified web notes directly into the analysis table.

Each candidate record must first be entered into `host_evidence_template.csv` with:

- a source identifier;
- the evidence type;
- the exact plant and bee name as reported;
- whether the observation concerns leaf cutting, nest lining, resin use, floral visitation, or only association;
- the geographic and temporal context when reported;
- a short verbatim evidence note or page/figure locator.

The formal plant universe is created only after taxonomic reconciliation and evidence grading.
